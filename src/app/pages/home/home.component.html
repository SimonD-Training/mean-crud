MongoDB CRUD Application

This application is designed to teach basic CRUD operations with MongoDB, ExpressJS, Angular, and NodeJS.

Features

This website has all the features of the CRUD operation using MongoDB, ExpressJS, Angular and NodeJS.

Create

The create action can be found at the create student page.

Read

The read operation is active on two pages the all students page and the student detail page which can be navigated to via the all students page.

Update

The update operation is active on edit student page which can be found by clicking a link on the all students page.

Delete

The delete operation is active on the all students page.
